# AMPROMOTORA

## Descrição

AMPROMOTORA é um site desenvolvido para uma agência de empréstimos, fornecendo informações sobre os serviços oferecidos, os tipos de empréstimos disponíveis e como os clientes podem entrar em contato.

## Funcionalidades Principais

-Página Inicial: Apresenta uma visão geral da agência de empréstimos, seus valores e serviços principais.
  
-Produtos e Serviços: Informações detalhadas sobre os diferentes tipos de empréstimos oferecidos, taxas de juros, prazos de pagamento, etc.

-Solicitação de Empréstimo: Formulário interativo para que os clientes possam preencher e solicitar um empréstimo diretamente pelo site.

-Calculadora de Empréstimos: Ferramenta que permite aos clientes calcular o valor das parcelas com base no valor do empréstimo, taxa de juros e prazo de pagamento.

-Perguntas Frequentes (FAQ): Seção dedicada a responder às perguntas mais comuns dos clientes sobre os serviços oferecidos.

-Contato: Informações de contato da agência, incluindo endereço, telefone e formulário de contato.

## Tecnologias Utilizadas

-PHP: Utilizado para o desenvolvimento do backend, processamento de formulários e interações com o servidor.

-HTML e CSS: Linguagens de marcação e estilo para o desenvolvimento da estrutura e do design do site.

-JavaScript: Utilizado para funcionalidades interativas e dinâmicas no frontend.

-Bootstrap: Framework front-end utilizado para o desenvolvimento responsivo e estilização dos componentes.

## Pré-requisitos

- Servidor web (como Apache ou Nginx) configurado para executar PHP.

## Instalação e Uso

1. Faça o clone deste repositório:
   ```
   git clone https://github.com/ArthurFarias11/AMPromtora.git
   ```

2. Configure seu servidor web para servir os arquivos PHP.

3. Acesse o site através do seu navegador web.
